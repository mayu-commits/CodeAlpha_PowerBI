Project Summary

This project is a collection of three interactive Power BI dashboards developed to demonstrate data analytics and visualization skills across different domains: finance, human resources, and education. The dashboards transform raw datasets into meaningful insights using KPIs, charts, and interactive filters to support data-driven decision-making.

The financial dashboard focuses on global sales performance and profitability trends, the HR dashboard analyzes employee performance and attrition patterns, and the student performance dashboard explores academic outcomes and influencing factors.

Together, these projects highlight practical skills in data visualization, business intelligence, and dashboard development using Power BI.
