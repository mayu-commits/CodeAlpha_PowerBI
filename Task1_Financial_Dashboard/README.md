# Global Financial Performance & Forecast Dashboard

##  Project Overview

This project presents an interactive **Global Financial Performance & Forecast Dashboard** built using **Microsoft Power BI**.
The dashboard provides insights into sales, profit, and product performance across different countries and customer segments.

The aim of this project is to help users quickly analyze financial performance and identify trends for better decision-making.

---

##  Key Metrics

The dashboard highlights the following important KPIs:

* **Profit Margin:** 0.14
* **Total Sales:** 118.73M
* **Total Units Sold:** 1.13M
* **Total Profit:** 16.89M
* **Average Selling Price:** 113.64

These indicators give a quick overview of overall business performance.

---

##  Dashboard Visualizations

### 1️ Sales Trend Over Time

Displays monthly sales trends to identify seasonal patterns and growth periods.

### 2️ Profit by Country

A treemap visualization showing profit distribution across different countries such as:

* France
* Canada
* Germany
* United States
* Mexico

### 3️ Sum of Profit by Product

A bar chart comparing profit generated by different products including:

* Paseo
* VTT
* Amarilla
* Velo
* Montana
* Carretera

### 4️ Sum of Sales by Segment

A pie chart illustrating sales contribution from different market segments:

* Government
* Small Business
* Enterprise
* Midmarket
* Channel Partners

---

##  Interactive Filters

The dashboard includes slicers that allow users to filter data by:

* Country
* Product
* Segment
* Year

These filters make the dashboard fully interactive and customizable for deeper analysis.
